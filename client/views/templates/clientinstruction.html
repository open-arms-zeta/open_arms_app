Instructions

Welcome Page

This is the welcome message. The meals you are ordering on the website are for the week starting on the date listed in the welcome message

These are your default meals. They are the meals in the current menu which correspond to the meal package which you are assigned to

If you are happy with your default options, click the "I Prefer the Default Options Listed Above" button. If you would like to customize your meals for the week, select the "I Would Like To Customize Meals" button

Meal Selection Page

This is a category header. You can click on this to view the meals in the menu for that category

Underneath each category header is a list of the meals for that category. When the page loads the list of meals for your default category will be visible. Clicking any of the other category headers will make those meals visible.

If you want to add a meal to your weekly selection, press the "Add to Meal" button to add it.

If you would like to go back to the main page, press the back button

Meal Selection Page (Continued)

This is the list of selected meal items. Meals will appear here as you add items form the menu. If you wish to remove a meal, press the "Remove Meal" button

When you are happy with your selection, press the "Submit Meal Selection" button to submit your custom meal pack

Confirmation Page

Once you have selected your meals, you will see a confirmation message containing the list of meals you ordered for the week. These selections are final and cannot be changed. You will not be able to select future meals until the following monday.

Inactive Page

If your account has been inactivated you will see this message, and will not be able to select a meal pack. Call client services for more information
